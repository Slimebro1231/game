# Chips Racing

A lightweight top-down isometric racing game with ghost replays and leaderboards.

## Play

Visit [chips.maxhanzhili.com](https://chips.maxhanzhili.com)

## Features

- **Arcade Racing**: Fast, responsive controls with drifting
- **Ghost Replays**: Race against your previous runs and top players
- **Leaderboards**: Submit your best times and compete globally
- **Lightweight**: Runs smoothly even on low-end devices

## Controls

| Key | Action |
|-----|--------|
| W / Arrow Up | Accelerate |
| S / Arrow Down | Brake / Reverse |
| A / Arrow Left | Steer Left |
| D / Arrow Right | Steer Right |
| Space | Handbrake (drift) |
| R | Reset Race |

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **Three.js** - 3D graphics
- **Cannon-es** - Physics (planned)
- **Vite** - Build tool
